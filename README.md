# President Simulator: 权力的游戏

## 简介
这是一个基于 Web 的政治与经济模拟策略游戏。玩家扮演一位新上任的总统，目标是在 4 年（48 个月）的任期内，在保证支持率不崩盘、不被暗杀的前提下，利用手中的权力和信息差，通过家族基金会积累尽可能多的财富。

## 游戏机制

### 1. 核心资源
*   **支持率 (Approval)**: 必须保持在 25% 以上，否则会被弹劾 (Game Over)。支持率决定了每回合恢复的行动点数 (AP)。
    *   >80%: 回复 8 AP
    *   >60%: 回复 6 AP
    *   >40%: 回复 4 AP
    *   <40%: 回复 2 AP
*   **资金 (Money)**: 初始资金因角色而异。目标是达到 $200 亿。资金低于 $1 亿会导致安保罢工而被暗杀 (Game Over)。
*   **行动点 (AP)**: 用于打出政治手牌。经济操作和技能不再消耗 AP。

### 2. 角色系统
游戏包含 6 位各具特色的角色，每位角色都有独特的初始资金、被动/主动技能：
*   **金发大亨**: 擅长操纵舆论，技能可直接大幅提升支持率。
*   **资深政客**: 精通幕后操作，技能提供额外 AP 并筛选特定卡牌。
*   **科技新贵**: 拥有内幕消息，技能可让下一次投资必胜且翻倍。
*   **退役将军**: 铁腕手段，技能可牺牲支持率换取免死金牌和军费。
*   **平权斗士**: 草根领袖，技能可将支持率转化为资金。
*   **好莱坞明星**: 粉丝经济，技能可提升支持率并提供无风险投资机会。

### 3. 政治卡牌系统
玩家通过打出手牌来治理国家。卡牌分为经济、军事、内政、外交、阴谋、关税等类型。
*   每张卡牌消耗 1-3 点 AP。
*   卡牌不仅影响支持率和资金，还会对 **全球经济** 和 **市场趋势** 产生深远影响。
    *   例如：发动贸易战会导致全球经济危机；呼吁降息会刺激股市牛市。
*   玩家每回合可抽取 2 张牌，手牌上限 6 张。可以花费 1 AP 弃掉不想要的牌。

### 4. 经济与市场系统 (核心玩法)
游戏模拟了一个受宏观经济影响的金融市场。
*   **三大市场**: 股市 (Stock)、加密货币 (Crypto)、大宗商品 (Commodity)。
*   **全球经济周期**: 经济处于 `增长` -> `繁荣` -> `衰退` -> `危机` 的循环中。政治行为会加速或改变这一进程。
*   **做多/做空**: 玩家可以对任意市场进行做多 (Long) 或做空 (Short) 操作。
*   **延迟结算**: 所有的投资行为都是“挂单”，在回合结束时，随着时间推进和事件触发，市场行情发生演变后，在下一回合初进行结算。这意味着玩家需要预判（或操纵）未来的市场走势。
*   **市场分数 (Market Score)**: 市场趋势由分数决定，允许多重利好/利空叠加，产生“超级牛市”或“市场崩盘”。

### 5. 随机事件与难度
*   **真实历史事件**: 包含互联网泡沫、次贷危机、加密寒冬、AI革命等基于 2000-2025 年真实历史改编的事件。
*   **自适应难度**: 随着玩家财富和支持率的增长，游戏会智能增加丑闻、调查、崩盘等负面事件的概率，保持后期挑战性。

## 技术栈
*   **前端框架**: Vue.js 3 (CDN 引入)
*   **样式**: CSS3 (Flexbox/Grid), Animate.css
*   **构建**: 原生 HTML/JS/CSS，无需编译，开箱即用。

## 移动端适配
游戏针对手机浏览器进行了深度优化：
*   **布局**: 采用底部导航栏 (Tab) 模式，分为“办公”和“市场”两个主要视图。
*   **交互**: 支持横向滑动选择角色，折叠式日志栏，紧凑的卡牌网格，以及触屏友好的大按钮设计。

## 如何运行
直接在浏览器中打开 `index.html` 即可开始游戏。推荐使用 Live Server 或类似工具以获得最佳体验。
